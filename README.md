# SundayBot

1. 썬데이 메이플을 검색해보기 귀찮은 유저들을 위한 디스코드 **SundayMapleBot**
2. - `!썬데이` 입력시 해당 주 썬데이 메이플 **링크** or **이미지** 출력
   - 업데이트 되지 않았을 시 `"아직 썬데이 메이플이 공개되지 않았어요!"` 출력
